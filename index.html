Reminder

Name Type Format Description
email string email Email id of the user
phonenumber string +919945055763 Phone number of the user (max 15 characters)
date string DD/MM/YYYY Date of reminder
time string HH:MM:SS Time of reminder
message string string Message for the reminder (max 250 characters)
timezone string time zone Defaults to Asia/Kolkata
